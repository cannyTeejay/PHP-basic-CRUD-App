# PHP File-Based CRUD Application

This is a simple **file-based CRUD application** built with **PHP**, **HTML**.  
The project demonstrates how to handle form input, persist data using files, and display stored data.

This project was created as a learning exercise to understand PHP fundamentals before working with a database.

---

## Features

- Create users using a HTML form
- Save user data to a text file
- Read and display stored user data
- Basic input validation and sanitization
- Simple project structure (no frameworks)

---

## Technologies Used

- PHP
- HTML
- File handling (`file_put_contents`, `file`, `explode`)
